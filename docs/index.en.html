What is Exam‑Excel?

Exam‑Excel is an exam training and self‑assessment system fully developed in Microsoft Excel using VBA.

It allows users to practice multiple‑choice questions, simulate real exams, review mistakes, and reinforce learning through immediate feedback and statistics.

The project is designed for certifications, competitive exams, and technical study, with a strong focus on transparency, reusability, and simplicity.

Main features

✅ Interactive exam training
✅ Random question selection
✅ Randomized answer order
✅ Session statistics
✅ Per‑question history
✅ Review priority logic
✅ Feedback and study notes
✅ Glossary and figures support

Basic workflow

  • Start a new session from the Principal sheet
  • Read the question in the Pregunta sheet
  • Answer by double‑clicking the selected option
  • Review feedback and continue

Excel file structure

  • Principal – session start and filters
  • Pregunta – main exam interface
  • BD – question database
  • Sesion – internal runtime state (hidden)
  • Glosario – technical terms
  • Gráficos – figures and images
